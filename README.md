## Important Rules

- Edit translated text values only. Do not change JSON keys.
- Keep placeholders such as `%s`, `%d`, `%u`, `{0}`, and `\n`.
- Keep color tags such as `<green>`, `<orange>`, `<red>`, `<blue>`, and `</green>`.
- Do not remove item names, skill names, NPC names, or map names unless you are correcting an official term.
- If you are unsure, submit feedback through the form instead of editing JSON directly.
